
## How to Start 

1. Clone the repository

2. Start XAMPP

3. Create database with name "bncc_project"

4. Start composer install
```
composer install
```

6. Start laravel local server
```
php artisan serve
```

5. Start migrations
```
php artisan migrate
```

6. Start seeders
```
php artisan db:seed
```

7. Go to [localhost:8000](localhost:8000)
